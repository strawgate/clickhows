# ClickHows

The Practical Almanac for Data &amp; Sundry Queries. A whimsical rebrand-as-vintage-almanac of the world's most practical database, served as a single static page.

Live: https://clickhows.com

## Stack

Single static `index.html`. No build step. Deployed to Cloudflare Pages.

## Local preview

```sh
python3 -m http.server 8080
# then open http://localhost:8080
```

## Deploy

Pushed to `main` → auto-deploys via Cloudflare Pages.
